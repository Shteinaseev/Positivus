# Positivus

## Описание проекта

Positivus - это демонстрационный веб-сайт, созданный на основе макета из Figma для демонстрации навыков верстки с использованием методологии БЭМ и препроцессора SCSS. Сайт включает в себя адаптивный дизайн, оптимизированный для различных устройств.

## Функциональность

- **Главная страница**: Героический баннер с призывом к действию
- **Услуги**: Раздел с описанием предоставляемых услуг
- **Команда**: Информация о членах команды
- **Отзывы**: Секция с отзывами клиентов
- **Контакты**: Форма обратной связи
- **Подписка**: Форма подписки на новости

## Технологии

- **HTML5**: Структура страницы
- **SCSS**: Стилизация с использованием препроцессора
- **БЭМ**: Методология для организации CSS классов
- **CSS**: Скомпилированные стили
- **JavaScript**: Интерактивные элементы (если применимо)

## Установка и запуск

1. **Клонируйте репозиторий:**
   ```
   git clone <url-репозитория>
   cd positivus
   ```

2. **Установите зависимости:**
   Убедитесь, что у вас установлен Sass:
   ```
   npm install -g sass
   ```

3. **Скомпилируйте SCSS в CSS:**
   ```
   sass styles/style.scss style.css
   ```
   Или используйте задачу в VS Code: "Compile SCSS to CSS"

4. **Запустите сайт:**
   Откройте `index.html` в вашем браузере или используйте локальный сервер:
   ```
   python -m http.server 8000
   ```
   Затем перейдите по адресу `http://localhost:8000`

## Разработка

- Для автоматической компиляции SCSS используйте:
  ```
  sass --watch styles/style.scss style.css
  ```
